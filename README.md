# Time Series Analysis: Singapore Births and Fertility Rates (1960-2024)

## Project Overview
This repository contains the data and R scripts for Assignment 1 of the Time Series Analysis course. The objective of this project is to perform an Exploratory Data Analysis (EDA) and fit initial time series models on Singapore's Total Live Births (TLB) and Total Fertility Rate (TFR) from 1960 to 2024.

## Repository Structure
* `data/`: Contains the raw dataset.
* `scripts/`: Contains the reproducible R scripts.
* `README.md`: Project documentation.
